# 📱 6-Digit Responsive OTP Input with Countdown Timer

A clean, modern, and SEO-friendly OTP (One-Time Password) verification component. Perfect for mobile apps and web platforms requiring 2FA or phone number verification.

## ✨ Features
- **6-Digit Support**: Pre-configured for standard verification codes.
- **Smart Navigation**: Automatic focus on next input & backspace to previous.
- **Arrow Keys**: Navigate between digits using Left/Right keyboard keys.
- **Live Timer**: Built-in "Code expires in" countdown functionality.
- **Mobile Ready**: Optimized `inputmode` for numeric keyboard popup.
- **SEO Optimized**: Lightweight and accessible code structure.

## 💻 Tech Stack
- **HTML5** (Semantic structure)
- **CSS3** (Flexbox layout)
- **jQuery** (DOM manipulation & Event handling)

## 🛠️ Customization
You can easily customize this OTP UI:
- Change OTP length (4 digit / 6 digit)
- Modify colors and fonts
- Connect to backend OTP API
- Add verification button
- Improve timer functionality

## 🚀 Installation & Setup
1. **Clone the repo**: `git clone https://github.com/chiragkumar8/responsive-otp-input-jquery-css`
2. **Include jQuery**: Ensure the CDN is in your `<head>`.
3. **Copy-Paste**: Drop the HTML/CSS/JS into your project and you're ready to go!
